# Macros

> **Status: Planned — next milestone**

Macros provide compile-time code generation and textual substitution, similar to C `#define` but with awareness of the language's type system.

## Intended Goals

- Named constants and expressions (replacing C `#define` constants)
- Conditional compilation (`#ifdef`-style, but cleaner)
- Code generation for repetitive hardware patterns
- Inline expansion of performance-critical sequences

Details will be defined in a later milestone.
